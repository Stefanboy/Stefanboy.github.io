rundeck的部署

  • 部署jdk

    rundeck要求jdk1.8，查看jdk的命令which jdk

  • 下载上传

    rundeck-launcher-2.10.1.jar

  • 运行停止命令

    java -jar rundeck-launcher-2.10.1.jar
后台运行
nohup java -jar rundeck-launcher-2.10.1.jar &
查询日志
tail -F nohup.out
停止命令
查看进程
ps -ef |grep rundeck
杀掉进程
kill -9 xxx
